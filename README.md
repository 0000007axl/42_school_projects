# 42 School Projects

A collection of my 42 School projects, combined into a single repository using `git subtree` to preserve each project's full commit history.

## Projects

| Project | Description |
|---------|-------------|
| [`libft`](./libft) | Custom C library reimplementing standard libc functions |
| [`ft_printf`](./ft_printf) | Custom implementation of `printf` |
| [`get_next_line`](./get_next_line) | Function that reads one line at a time from a file descriptor |
| [`push_swap`](./push_swap) | Sorting algorithm using two stacks with a limited set of operations |

## Structure

Each project's complete commit history is preserved in this repository under its respective directory, viewable with `git log --follow <file>` or `git log <directory>/`.
